## Overview

The Backend API is a Django-based service that powers our job board and recruitment platform. It provides robust endpoints for managing the entire lifecycle of job postings, from creation to publication on various talent channels, campaign management, and candidate application tracking.

## Getting Started

To get up and running with the Backend API locally, follow our [Quickstart Guide](/quickstart).

## Key Features

- **Campaign Management**: Group job bookings into promotional campaigns to optimize reach and track performance.
- **Integrations & Webhooks**: Built-in support for major platforms including Indeed Apply, StepStone, Perspective, and Facebook.
- **Magic Links**: Secure, token-based authentication for passwordless access to company and application resources.
- **Internal Admin Panel**: Comprehensive internal endpoints and Django admin interface for platform operators.
