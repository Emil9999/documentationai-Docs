## Step 1: Authenticate

To interact with the Kini API, you need to authenticate using an **Bearer API Token**. Follow the steps in the [Authentication](/kini-api-basics/authentication) to:

1. Obtain an API Key in the [Kini Partner App](https://app.getkini.com/).
2. Include the key in the Authorization header for all API requests.

***

## Step 2: Explore Endpoints

The Kini API provides robust tools for managing jobs, candidates, and applications. Explore the following guides to learn more about specific features:

* [Jobs API](/api/jobs/list): Manage job postings, including creation, updates, and synchronization.
* [Applications API](/api/applications/list): Track application statuses and submit new candidates for jobs.

***

## Step 3: Best Practices

Make the most of the Kini API by following these best practices:

* **Secure Tokens**: Keep access and refresh tokens secure. Rotate tokens regularly as described in the [Authentication Guide](/kini-api-basics/authentication).
* **Use Pagination**: Optimize data handling by enabling pagination. See the [Pagination Guide](/kini-api-basics/pagination) for details.
* **Handle Errors Gracefully**: Refer to the [Error Handling Guide](/kini-api-basics/error-handling) for common scenarios and troubleshooting steps.
