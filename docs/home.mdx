## Quick start

<Steps>
  <Step title="Get an API key" icon="key">
    Sign in to the [Kini Partner App](https://app.getkini.com/) and generate a Bearer API key.
  </Step>
  <Step title="Make your first request" icon="play">
    Call the [List Jobs](/docs/api/jobs/index) endpoint with your API key in the `Authorization` header.
  </Step>
  <Step title="Subscribe to webhooks" icon="webhook">
    Configure webhooks in the Partner App to receive real-time updates when jobs or applications change.
  </Step>
</Steps>
